# 冒烟测试清单（当前 MVP）

在微信开发者工具中验证以下路径。云试戴需已部署 `tryon` 并配置 `DASHSCOPE_API_KEY`。

## 1. 首页与款式展示

1. 编译 → 登录（或跳过）→ 首页
2. 「为你推荐」4 张卡片应显示**真实款式封面**（美团 CDN），非 emoji
3. 进入「热款榜」→ TOP 列表封面与款式名称一致

## 2. 云试戴（核心）

1. 首页 → **试戴** → 选一款 → 开始试戴
2. 手照步骤：
   - `USE_MOCK_HAND_PHOTO: true` → 点「使用默认手照」或任选 **评测手照**
   - 预览应**完整显示**手照（widthFix，不裁剪）
3. 等待 30–90s → 结果页出现 AI 合成图
4. 控制台无 `[tryon-cloud]` 致命错误

**云函数 ping**（云开发控制台测试）：

```json
{"action":"ping"}
```

期望：`hasDashScopeKey: true`，`wanModel: "wanx2.1-imageedit"`

## 3. 款式库

1. Tab → 款式库 → 列表为 25 条真实数据
2. 筛选 / 搜索 → 详情页封面与字段正常
3. 收藏 → 「我的 - 收藏」可见

## 4. 商家 / 预约（Mock）

1. 我的 → 商家入口 → 看板数据可加载（Mock）
2. 轻预约配置可保存（本地 Mock）

## 5. 已知非 MVP

以下页面存在代码但未接入主流程，**不要求**本次通过：

- AR 试戴（`try-on-ar`）
- AI 同款（`ai-match`）
- 甲型测量（`nail-shape`）

## 验收标准

- [ ] 首页 + 热榜真实封面
- [ ] 云试戴至少 1 次成功出图
- [ ] 款式库 / 详情 / 收藏正常
- [ ] 无阻塞性 JS 异常
